Detailed Description

Implements IAllocator using the C dynamic memory allocation (malloc, realloc and free) This implementation wraps malloc, realloc and free. These functions don't not handle custom alignment natively, so extra space equal to the alignment is requested to malloc. A native support for alignment would avoid this waste of space.
